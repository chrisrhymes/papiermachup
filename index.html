PapierMachUp

Welcome to PapierMachUp. A really simple multi column mobile first layout stylesheet.

Use it to quickly mock up different layouts for your mobile first website

Its really easy to use with three main options.

  • full-width provides an always full width container
  • column provides a single column below 700px and two columns above
  • sub-column provides two columns below 700px and four columns above

Like I said, really simple.

An example

The below example is constructed from two sub-column and one column:


    <div class="sub-column">
        <p>Sub-Column</p>
    </div>
    <div class="sub-column">
        <p>Sub-Column</p>
    </div>
    <div class="column">
        <p>Column</p>
    </div>
